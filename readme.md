# 🖼️ Image Processor (C++)

Проект представляет собой консольное приложение для обработки изображений в формате BMP.  
Реализованы базовые фильтры и работа с бинарным форматом изображений.

## 🚀 Возможности

- Чтение и запись BMP-файлов (24-bit, без сжатия)
- Применение фильтров к изображениям:
  - Grayscale (оттенки серого)
  - Negative (негатив)
  - Sharpening (резкость)
  - Edge Detection (выделение границ)
  - Crop (обрезка)
  - Gaussian Blur (размытие)
- Поддержка цепочек фильтров (pipeline)

---

## 🛠️ Технологии

- C++17
- Работа с бинарными файлами
- Модульная архитектура (разделение на reader/writer, filters, core)
- Обработка ошибок через исключения

---

## 🧠 Архитектура

Проект разделён на несколько ключевых компонентов:

- **bmp_reader / bmp_writer** — чтение и запись BMP-файлов  
- **image / color** — базовые структуры изображения  
- **filters/** — реализация фильтров  
- **filter_factory** — создание фильтров по параметрам  
- **main / cli** — обработка аргументов командной строки  

Такое разделение упрощает добавление новых фильтров и расширение функциональности.
